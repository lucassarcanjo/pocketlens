Contribution guide lives at [`docs/contributing.md`](docs/contributing.md).

For quick reference:

```sh
brew install xcodegen
make gen
open app/PocketLens.xcodeproj
```

Run `make test` before opening a PR.

See [`docs/architecture.md`](docs/architecture.md) for the module layout and [`.claude-plans/00-OVERVIEW.md`](.claude-plans/00-OVERVIEW.md) for the phased roadmap.
